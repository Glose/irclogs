# irclogs
#### A logger for #laravel, built with Laravel and MongoDB

http://irclogs.laravel.io

---


```bash
$ composer install
$ bower install
$ forever start main.js
```

### Sample log data

```bash
$ php artisan db:seed
```

### Compile CSS

```bash
$ compass compile
```
