# irclogs
#### A logger for #laravel, built with Laravel and MongoDB

http://irclogs.laravel.io

---

![Screenshot](http://f.cl.ly/items/0D110U140e3o3c1t3L1m/Screenshot%202013-07-24%20%C3%A0%2017.08.03.png)

### Install

```bash
$ composer install
$ bower install
$ forever start main.js
```

### Sample log data

```bash
$ php artisan db:seed
```

### Compile CSS

```bash
$ compass compile
```
